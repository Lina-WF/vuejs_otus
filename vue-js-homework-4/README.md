# Четвёртая домашняя работа

## 1. Поиск в шапке страницы
Был реализован поиск по названию-описанию-категории и по цене в формате "от" и "до".

## 2. Форма отправки заказа, её валидация и отправка на эхо-сервер
Были реализованы две формы заказа (информация о доставке и информация о карте). После нажатия на кнопку "В корзину" отображается форма доставки. После её заполнения отображается форма карты. После её заполнения отображается ответ сервера.

Валидация форм реализована через VeeValidate. Перейти к следующей форме невозможно без корректно пройденного предыдущего этапа.

Информация с двух форм собирается, объединяется и отправляется на эхо-сервер (https://httpbin.org/). После ответа сервера выводится успешность выполнения.

## 3. Загрузка данных
Данные загружаются из json-файла серез fetch

## 4. Форма добавления нового товара
Реализована форма добавления нового товара. Форма отображается после нажатия на кнопку "Добавить новый товар". Валидация формы реализована через VeeValidate. После заполнения формы формируется объект, который добавляется в общий массив товаров.
